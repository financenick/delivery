## Ёбидоёби — фронтенд доставки суши и роллов

Одностраничное приложение, повторяющее дизайн и основную функциональность сайта [`izhevsk.yobidoyobi.ru`](https://izhevsk.yobidoyobi.ru/):

- шапка с логотипом, выбором адреса, телефоном и поиском;
- навигация по категориям (наборы, роллы и суши, ёнигири, темпура и т.д.);
- сетка карточек блюд с весом, описанием и ценой;
- корзина с подсчётом количества позиций и общей суммы, изменением количества и очисткой;
- адаптивная вёрстка для десктопа и мобильных устройств.

Реализовано как SPA на React + TypeScript + Vite.

### Установка и запуск

```bash
npm install
npm run dev
```

После запуска открой в браузере адрес из консоли (обычно `http://localhost:5173`).

### Сборка продакшн-версии

```bash
npm run build
npm run preview
```
